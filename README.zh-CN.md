# AI工具集


## 快速开始

### 用法

#### SDK

安装：

```sh
pnpm add aikit@latest

```

使用：

```ts
import { generateText } from 'aikit/text'
import { generateImage } from 'aikit/image'
import { generateVideo } from 'aikit/video'

```


#### CLI

```sh
pnpm add -g aikit@latest

```
